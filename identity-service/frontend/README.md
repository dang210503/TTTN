# Frontend - Hệ Thống Quản Lý Phòng Khám

## Cài đặt

```bash
cd frontend
npm install
```

## Chạy ứng dụng

```bash
npm run dev
```

Ứng dụng sẽ chạy tại: http://localhost:3000

## Tính năng

- ✅ Quản lý bệnh nhân (User) với vai trò: Bệnh nhân, Bác sĩ, Nhân viên
- ✅ Quản lý thuốc/dịch vụ (Product) với cảnh báo hết hạn
- ✅ Quản lý lịch khám (Appointment) với liên kết bệnh nhân và bác sĩ

## Yêu cầu

- Node.js 16+
- Backend đang chạy tại http://localhost:8080
